# 简介

## 存放路径

所有物品配置文件应存放于`plugins/NeigeItems/Items`文件夹

重复 ID 的物品仍然会被加载，但可能互相覆盖

## 编写你的物品

### /ni save是万物起源

遇事不决，/ni save。如果不行，就/ni cover。这是最简单最便捷的快速生成物品配置的方法
